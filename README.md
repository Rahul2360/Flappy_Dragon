## Flappy_Dragon
# About game
This game is created with the help of UNITY. 
I used C# programmming while preparing this game.
This is an android game.

# How to download
There is a file named "Flappy Dragon.apk". Download this file and install in the android mobile .

# How to Play
There is a dragon in the game and there are obstacles in the path of the dragon.
We have to fly the dragon by tapping on the screen and do not touch the dragon with the obstacles .
